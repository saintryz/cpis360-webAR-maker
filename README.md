# CPIS 360 - Marker-Based WebAR Lab 1

This project is a simple marker-based WebAR application for CPIS 360 (Introduction to Immersive Technologies).  
When the camera detects the Hiro marker, 3D shapes and text appear above the marker.

## Tools Used

- A-Frame
- AR.js
- HTML5 + CSS
- GitHub Pages (HTTPS hosting)

## How to Run

1. Open `index.html` locally (desktop for quick check), or
2. Deploy the project to GitHub Pages for mobile testing (recommended for assignment).
3. On mobile, open the hosted HTTPS link, allow camera access, and point to the Hiro marker.
